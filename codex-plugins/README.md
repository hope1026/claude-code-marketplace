# Codex Plugins

Codex skill collection for distribution.

## Structure

- `.agents/skills/`: official discovery path for custom Codex skills in this repo
- `AGENTS.md`: repository guidance for working on this package

## Included Skills

- `ai-council`
- `claude-cli`
- `gemini-cli`
- `review-ralph`
- `task-plan`
- `task-ralph`

## Usage

Copy or sync the skill folders under `.agents/skills/` into either:

- a project's `.agents/skills/`
- your user-level Codex skills directory
